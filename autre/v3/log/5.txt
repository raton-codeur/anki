les versions HTTP :

● 1991 → HTTP/{{c1::0.9}} :
juste GET, pas de header / body, pas de status code,
le serveur renvoie du HTML puis ferme la connexion

● 1996 → HTTP/{{c1::1.0}} :
headers (Content-Type, Content-Length...), status code,
une requête = une connexion (par défaut mais keep-alive possible)

● 1999 → HTTP/{{c1::1.1}} :
keep-alive par défaut, headers obligatoires (host, content-length / transfert-encoding: chunked)
possible : chunked transfer encoding, pipeline,
meilleure gestion des caches

● 2015 → HTTP/{{c1::2}} :
plus optimisé : multiplexage, compression des headers, format binaire, priorisation des requêtes

● 2022 → HTTP/{{c1::3}} :
UDP à la place de TCP, moins de latence
