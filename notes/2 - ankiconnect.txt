envoyer des requetes http json en Python
	fonction ankiconnect
		import requests

		payload = {
			"action": action,
			"version": VERSION,
			"params": params or {}
		}

		r = requests.post("http://127.0.0.1:8765", json=payload)
			"r" -> "response"

			ça lève requests.exceptions.ConnectionError si erreur

		data = r.json()
			data["result"]
				ce qu'on renvoie
			data["error"]
				si c'est pas null
					on arrête le programme

la liste de notes

	une note

		● "deckName"

			le nom du type de note
